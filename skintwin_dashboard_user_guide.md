# SkinTwin-ASI Dashboard User Guide

## Introduction

The SkinTwin-ASI Dashboard provides a comprehensive visualization of the SkinTwin Architecture, including dynamic flows and membrane nesting topologies. This user guide explains how to use the dashboard effectively to explore and understand the architecture, flows, and membrane structures.

## Dashboard Overview

The dashboard consists of five main sections:

1. **Top Control Panel**: Navigation, visualization mode selection, search, and settings
2. **Left Sidebar**: Component hierarchy, layer controls, and bookmarks
3. **Central Visualization Area**: Main interactive visualization display
4. **Right Sidebar**: Detailed information about selected components, flows, or membranes
5. **Bottom Panel**: Timeline and simulation controls

## Getting Started

### Accessing the Dashboard

1. Open the SkinTwin-ASI application
2. Navigate to the Dashboard section
3. The dashboard will load with the default Architecture view

### Understanding the Interface

The dashboard is designed to provide multiple perspectives on the SkinTwin-ASI architecture:

- **Architecture View**: Shows the structural components and their relationships
- **Flow View**: Visualizes dynamic flows between components
- **Membrane View**: Displays membrane nesting hierarchies
- **Combined View**: Integrates architecture, flows, and membranes in a layered visualization

## Visualization Modes

### Architecture View

The Architecture View focuses on the structural components of the SkinTwin-ASI system:

1. **Application Layer**: Flow Designer, Analysis Studio, Intervention Planner, Membrane Modeler
2. **Framework Layer**: Relevance Realization Framework, Triadic Polarities, Axis Mundi Alignment
3. **Agent Layer**: Deep Tree Echo, Marduk, PLingua Membrane, JAX CEO

To use the Architecture View:
- Select "Architecture" in the top control panel
- Navigate the component hierarchy in the left sidebar
- Click on components to view their details in the right sidebar

### Flow View

The Flow View visualizes the dynamic flows between components:

1. **Data Flows** (Blue): Transfer of information, data, and content
2. **Control Flows** (Red): Commands, directives, and orchestration signals
3. **Feedback Flows** (Green): Performance metrics, error signals, and learning information
4. **Meta-Cognitive Flows** (Purple): Self-awareness, reflection, and adaptation signals

To use the Flow View:
- Select "Flows" in the top control panel
- Use the flow type filters in the left sidebar to show/hide specific flow types
- Click on flows to view their details in the right sidebar
- Observe flow animations that represent latency, bandwidth, and reliability

### Membrane View

The Membrane View displays the nested membrane hierarchies:

1. **Biological Membranes** (Green): Skin layers, cells, organelles
2. **Computational Membranes** (Blue): P-system structures, transition membranes, division membranes
3. **Conceptual Membranes** (Purple): Knowledge domains, cognitive boundaries, functional separations

To use the Membrane View:
- Select "Membranes" in the top control panel
- Use the membrane type filters in the left sidebar to show/hide specific membrane types
- Click on membranes to expand/collapse their nested structures
- Click on membrane objects or rules to view their details

### Combined View

The Combined View integrates architecture, flows, and membranes in a layered visualization:

To use the Combined View:
- Select "Combined" in the top control panel
- Use the layer controls in the left sidebar to show/hide specific layers
- Adjust layer opacity if needed
- Use all selection and filtering options as in individual views

## Interactive Features

### Navigation and Exploration

#### Zoom and Pan
- Use the mouse wheel to zoom in and out
- Click and drag to pan the visualization
- Use the zoom controls in the central visualization area to reset the view

#### Component Hierarchy Navigation
- Expand/collapse nodes in the component hierarchy tree in the left sidebar
- Click on components in the hierarchy to navigate to them in the visualization

### Selection and Filtering

#### Component Selection
- Click on a component in the visualization to select it
- View detailed information about the selected component in the right sidebar

#### Flow Filtering
- Use the flow type checkboxes in the left sidebar to show/hide specific flow types
- Filter flows by source or target component by selecting components

#### Membrane Filtering
- Use the membrane type checkboxes in the left sidebar to show/hide specific membrane types
- Expand/collapse membrane hierarchies by clicking on membranes

### Details Panel

The right sidebar displays detailed information about the selected item:

#### Component Details
- Name and type
- Position and size
- Function and description
- Status and performance metrics
- Connections to other components

#### Flow Details
- Type and label
- Source and target components
- Intensity and state
- Latency, reliability, and bandwidth metrics

#### Membrane Details
- Name and type
- Charge state
- Parent membrane
- Objects contained within the membrane
- Rules associated with the membrane

### Timeline Controls

The bottom panel contains timeline and simulation controls:

- **Playback Controls**: Play, pause, step forward/backward, reset
- **Timeline Scrubber**: Navigate to specific points in time
- **Simulation Speed**: Adjust the speed of animations and simulations
- **Status Information**: Current time, active processes, and other metrics

## Advanced Features

### Bookmarks

Save and load specific views for quick access:

1. Navigate to the desired view and configuration
2. Click "+ Add Current View" in the Bookmarks section of the left sidebar
3. Enter a name for the bookmark
4. Click on saved bookmarks to restore their views

### Layer Controls

Customize the visualization by showing/hiding specific layers:

- Architecture Layer: Structural components and relationships
- Flow Layer: Dynamic connections between components
- Membrane Layer: Nested membrane structures

### Global Search

Search for specific components, flows, or membranes:

1. Enter search terms in the global search box in the top control panel
2. Results will highlight matching items in the visualization
3. Click on search results to navigate to them

## Troubleshooting

### Visualization Issues

If visualizations are not displaying correctly:
- Try refreshing the dashboard
- Check that all layer visibility toggles are enabled
- Reset the zoom level to default
- Ensure your browser is up to date

### Performance Issues

If the dashboard is running slowly:
- Reduce the number of visible flows and membranes using filters
- Switch to individual visualization modes instead of combined view
- Close other browser tabs and applications

### Selection Issues

If selection is not working as expected:
- Try clicking directly on the center of components or flows
- Ensure the item is currently visible (not filtered out)
- Reset the visualization by refreshing the dashboard

1. Start with the Architecture View to understand the overall structure

1. Use the Flow View to focus on dynamic connections

### Exploring Membrane Structures

1. Use the Membrane View to focus on nested hierarchies
2. Start with top-level membranes and gradually expand nested structures
3. Explore objects and rules within membranes
4. Compare different membrane types (biological, computational, conceptual)

### Using Combined View

1. Start with a specific focus area or component
2. Gradually add layers as needed
3. Use filtering to reduce visual complexity
4. Save useful configurations as bookmarks
